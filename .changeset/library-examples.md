Add example scripts for use-m, lino-arguments, and command-stream libraries

Added four new example scripts demonstrating how to use link-foundation libraries:

- `examples/use-m-example.mjs` - Dynamic npm package loading without package.json
- `examples/lino-arguments-example.mjs` - Unified configuration from CLI/env/.lenv
- `examples/command-stream-example.mjs` - Concise CLI command execution
- `examples/combined-libraries-example.mjs` - All three libraries working together
